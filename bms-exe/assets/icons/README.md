# Icons

Place your app icons here:

## macOS
- Place your `.icns` file in `mac/` folder and rename it to `icon.icns`

## Windows
- Place your `.ico` file in `win/` folder and rename it to `icon.ico`

## Linux/PNG
- Place your PNG files (1024x1024 recommended) in `png/` folder

## Creating Icons

1. Create a 1024x1024 PNG icon
2. For macOS: Use Image2icon or similar tool to create `.icns` file
3. For Windows: Use iConvert Icons or similar tool to create `.ico` file
